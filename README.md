# CSMatic Website

Website for CSMatic - AI-Powered Customer Service Agent that handles customer inquiries 24/7, reduces costs, and delivers exceptional support experiences.

## Project Structure

- `src/`: Contains website source code
  - `assets/`: Stores static assets like images and SVGs
  - `components/`: Astro components used throughout the website
  - `content/`: Blog and help content
  - `layouts/`: Base layouts for pages
  - `pages/`: Website pages
  - `styles/`: CSS and Tailwind files
- `public/`: Static files that will be copied to the build directory
- `astro.config.mjs`: Astro configuration
- `tailwind.config.mjs`: Tailwind CSS configuration

## Technologies

- [Astro](https://astro.build/) - Modern web framework
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [@tailwindcss/typography](https://tailwindcss.com/docs/typography-plugin) - Tailwind plugin for content styling

## Website Features

- Landing page with product explanation
- Pricing page
- FAQ page
- Help page
- Blog
- Privacy policy and terms of service pages

## Development

### Prerequisites

- Node.js (latest version)
- npm or yarn

### Installation

```bash
npm install
```

or

```bash
yarn
```

### Running Development Server

```bash
npm run dev
```

or

```bash
yarn dev
```

Development server will run at `http://localhost:4321`

### Build for Production

```bash
npm run build
```

or

```bash
yarn build
```

### Preview Build

```bash
npm run preview
```

or

```bash
yarn preview
```
